Detailed Description

This is a father class to all MQSensor types, it is not intended to be instanciated, but can be used as a pointer array if your code needs so, also can be used to instanciate and use multiple types of MQ sensor in the same object.

float const MQSensor::MQResistanceCalculation ( const float  raw_adc)
inlineprotected

Returns the tension on the sensor, solving the voltage divider for R1 as seen on https://en.wikipedia.org/wiki/Voltage_divider. The tension is given by a voltage divider, which has its load resistor set in _RL_VALUE, and can be adjusted in code if the sensor comes before or after the load resistor.

Parameters
raw_adcThe ADC value of the voltage divider on the _MQ_pin.
Returns
The resistance of the MQ sensor.

float MQSensor::MQTension ( const float  raw_adc)

Returns the voltage either in the load resistor, or in the MQ sensor, either one that's between the MQSensor::_MQ_pin and the ground connection.

This is usually used as a part of the MQSensor::MQGetPPM function.

Parameters

Returns
The tension value remapped to a float value between 0 and 5 volts.
